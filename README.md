# DeFi Yield Tracker

A simple tool that tracks and compares yield opportunities across decentralized finance (DeFi) protocols.

## Features
- Track yield rates across DeFi platforms
- Compare APY between multiple protocols
- Monitor liquidity pool returns
- Educational tool for learning DeFi analytics

## Technologies
- Python
- Web3 data integration
- DeFi APIs
- REST API

## Purpose
This project was created to explore decentralized finance data and understand how yield opportunities across DeFi protocols can be tracked and analyzed.

## Future Improvements
- Support for more DeFi protocols
- Real-time yield updates
- Historical yield analytics
- Integration with DeFi portfolio trackers
